# 🧠 Assest: The Company Brain

**Assest** is a high-precision agentic knowledge ingestion and retrieval engine. It transforms fragmented organizational data into a unified, version-aware "Company Brain" capable of self-learning and expert-level reasoning.

## API Endpoints

- **Health Check**: `GET /health/live`
- **Query**: `POST /api/query`
- **Stream**: `POST /api/query/stream`
- **Connectors**: `GET /api/connectors`
